

## Описание проекта

Инструмент для визуализации графа зависимостей пакетов Maven. Программа анализирует зависимости указанного пакета и создает графическое представление в форматах Mermaid и SVG без использования готовых менеджеров пакетов или библиотек для получения зависимостей.

## Функциональность

### Этап 1: Конфигурация
- Чтение параметров из XML-файла конфигурации
- Валидация и вывод всех параметров в формате ключ-значение
- Обработка ошибок конфигурации

### Этап 2: Сбор данных
- Получение информации о прямых зависимостях из Maven-репозитория
- Парсинг POM-файлов для извлечения зависимостей
- Вывод списка зависимостей в консоль

### Этап 3: Визуализация
- Генерация текстового представления графа на языке Mermaid
- Создание SVG-изображения графа зависимостей
- Автоматическое масштабирование под содержимое

### Запуск
- python Kod2.py config.xml
